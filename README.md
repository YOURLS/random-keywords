Random Keywords
===============

Plugin for [YOURLS](http://yourls.org) `1.5` to `1.7.3` (this plugin is bundled with YOURLS 1.7.4+)

Description
-----------
Assign random keywords to shorturls, like bitly (ie http://sho.rt/hJudjK).

Installation
------------
1. In `/user/plugins`, create a new folder named `random-keywords`.
2. Drop these files in that directory.
3. Go to the Plugins administration page ( *eg* `http://sho.rt/admin/plugins.php` ) and activate the plugin.
4. Have fun!
